## Dataset

This project uses historical air quality data collected from structured CSV files and additional tabular web data that can be scraped from HTML pages. The dataset contains a large number of observations and includes measurements related to air quality across different cities, countries, and time periods.

Key variables in the dataset include:
- Air Quality Index (AQI)
- PM2.5
- PM10
- NO2
- SO2
- CO
- O3
- Temperature
- Humidity
- Wind speed

These variables support the analysis of pollution levels, geographic variation, and seasonal trends in air quality.

## Problem Set Tasks
1. Scrape additional air quality data from web sources using Beautiful Soup.
2. Integrate the scraped data with the existing CSV datasets.
3. Perform statistical analysis, including mean, variance, and percentiles, using NumPy.
4. Create time-series visualisations with trend lines and confidence intervals.
5. Build a data pipeline that automates the full process.
